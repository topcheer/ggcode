# GGCode Beta Access — GUI, TUI & Mobile

Hi everyone,

We're opening beta access to GGCode across all platforms. Here's a quick overview of how each client works.

**Website:** https://ggcode.dev
**GitHub:** https://github.com/topcheer/ggcode

---

## Desktop GUI

GGCode Desktop is a native macOS/Windows/Linux app built with Fyne. It provides a full graphical interface for the AI coding agent:

- Chat with the AI agent in a rich markdown panel
- View tool execution results inline (file edits, terminal commands, git operations)
- Monitor sub-agent and teammate activity in real-time
- **Share your session**: Click the Share button in the sidebar to generate a QR code that mobile devices can scan to connect remotely

The GUI app installs and runs alongside the TUI — you can use either or both.

## Terminal TUI

GGCode TUI is the terminal-based interface — the original way to interact with the agent:

- Full-featured chat interface in your terminal with streaming markdown rendering
- Slash commands for everything: `/share` or `/tunnel` to generate a mobile QR code, `/sessions`, `/memory`, `/skills`, etc.
- Works over SSH — ideal for remote development
- Incremental rendering engine for smooth streaming output

Both GUI and TUI connect to the same agent backend, so you can switch between them freely.

## Mobile App

GGCode Mobile is a companion app that lets you interact with your desktop agent from your phone:

- **Connect**: Scan the QR code generated by Desktop GUI (Share button) or TUI (`/share` command)
- **Chat**: Send messages to the AI agent in real-time
- **Review**: Approve or reject tool executions on the go
- **Monitor**: Track sub-agent tasks, build status, and code changes from anywhere
- Auto-reconnects in the background — stays connected even when you switch apps

The mobile app does not run an agent locally. It's a remote control for your desktop GGCode session.
