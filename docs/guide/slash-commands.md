# Slash Commands

Slash commands are typed directly in the chat input within the TUI.

## Commands

| Command | Description |
|---------|-------------|
| `/help` | Show help |
| `/update` | Check and install updates |
| `/sessions` | List and resume sessions |
| `/compact` | Compact conversation to save context |
| `/clear` | Clear conversation |
| `/mode [default\|bypass\|readonly\|yolo]` | Switch permission mode |
| `/cost` | Show token usage and cost |
| `/mcp` | Show MCP server status |
| `/im` | Show IM adapter status |
| `/cancel` | Cancel current operation |
| `/exit` | Exit ggcode |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Tab` | Switch between chat and tools panes |
| `Ctrl+C` | Cancel current operation |
| `Ctrl+L` | Clear screen |
