install

Pick your path.
Get running in seconds.

macOS / Linux

No sudo required. Installs to ~/.local/bin

curl -fsSL https://ggcode.dev/install.sh | bash

Windows

No admin required. Uses winget or PowerShell

winget install gg.ai.ggcode-cli

Homebrew

macOS and Linux via Homebrew tap

brew install topcheer/ggcode/ggcode

npm

Node ecosystem, installs release binary

npm install -g @ggcode-cli/ggcode

pip

Python environments or team distribution

pip install ggcode-cli
