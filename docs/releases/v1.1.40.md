# ggcode v1.1.40

## Fixes

- **Windows CI smoke test: harden against Defender scanning** — Each smoke command now runs under a per-command timeout (30 s / 15 s) via `Start-Process -Wait` instead of inline execution. If Defender blocks a command, the job logs a warning instead of hanging for 5 minutes. The job-level timeout is reduced to 3 minutes.
