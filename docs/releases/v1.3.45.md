# v1.3.45

## Features

### Mobile UX Improvement

- **Tap to dismiss keyboard**: Tapping anywhere in the message area now dismisses the on-screen keyboard, making it easier to review chat history while typing.
- **Scroll to dismiss**: Dragging/scrolling the message list also dismisses the keyboard via `ScrollViewKeyboardDismissBehavior.onDrag`.
