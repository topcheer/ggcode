# v1.2.2

## Swarm/Team Messaging Improvements

**send_message now async**
- Sending a message to a swarm teammate returns immediately instead of blocking for result (previously up to 5 minutes)
- LLM retrieves teammate execution results asynchronously via `teammate_results` tool
- Sub-agent path was already async, no changes needed

**Teammate execution timeout removed**
- Previous 5-minute timeout on teammate execution removed
- Timeout is now controlled by the agent loop's `max_iterations` setting
- Teammates working on complex tasks no longer cut off prematurely

**Task board improvements**
- `swarm_task_create` now supports `assignee` field for direct task assignment
- Tasks with assignee are delivered directly to teammate inbox for immediate execution
- `send_message` documented as for unstructured follow-ups, not task assignment

## Other

- `todo_write` tool description strengthened: requires status updates after creation
