# v1.2.2

## Swarm/Team 消息机制改进

**send_message 改为异步发送**
- 发消息给 swarm teammate 后立即返回，不再阻塞等待结果（之前最多等 5 分钟）
- LLM 通过 `teammate_results` 工具异步获取 teammate 的执行结果
- sub-agent 路径本来就是异步的，无需改动

**teammate 执行超时去除**
- `TeammateTimeout` 默认从 30 分钟改为 0（无超时），复杂任务可以运行数小时不受限制
- 用户仍可通过 `swarm.teammate_timeout` 配置项显式设置超时上限

**工具描述更新**
- `send_message`：明确说明 "Messages are sent asynchronously — the tool returns immediately"
- `teammate_results`：强调 "Use this after send_message to check if a teammate has finished"

**crash 修复**
