# v1.3.47

## Features

### save_memory Live Prompt Refresh

When `save_memory` is called, the agent system prompt is now rebuilt immediately to include the newly saved memory. Previously, saved memories only took effect on the next session or agent restart. This works across all three modes: TUI, daemon, and pipe.

### Paste Shortcut Hints

All text input fields across the application now display a paste shortcut hint in the placeholder:
- **macOS**: `Cmd+V`
- **Windows**: `Ctrl+V`
- **Linux**: `Ctrl+Shift+V`

Covered areas:
- All 15 IM adapter panels (Telegram, Discord, QQ, Slack, DingTalk, Feishu, Matrix, IRC, Signal, WhatsApp, Nostr, Mattermost, Twitch, WeCom, PC)
- MCP panel, Provider panel
- Harness panel
- `ask_user` questionnaire inputs
- Onboard wizard (all 5 input fields)

Bilingual support (English / zh-CN).

### Desktop Native Titlebar (macOS)

The macOS desktop app now uses an integrated native titlebar with custom chrome:
- Traffic light buttons (close/minimize/maximize) are properly positioned
- Custom title label shows workspace name and model
- Theme-aware coloring that updates on theme change
- Linux and Windows continue using the standard titlebar
