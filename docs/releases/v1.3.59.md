# v1.3.59

Desktop UI fixes and CI stability improvements.

## Desktop

### Windows Icon and App Name
- Embedded `.syso` resource file into Windows `.exe`; taskbar/explorer/shortcut now correctly shows GGCode icon
- EXE metadata includes ProductName, FileDescription, CompanyName
- Fyne app ID changed from `com.ggcode.desktop` to `gg.ai.ggcode-desktop`; Windows Task Manager now shows "GGCode Desktop"

### Markdown Code Block Improvements
- Syntax highlighting now works for Go, Python, JavaScript, TypeScript, Bash, JSON, YAML, HTML, CSS, and XML
- Added line numbers to code blocks (toggleable)
- Code blocks auto-wrap long lines instead of truncating
- Inline code uses monospace font with background highlight

### Session Switch Optimization
- Session list loading moved to background thread; UI no longer freezes with many sessions
- Session switch uses `fyne.CurrentApp().Send()` to ensure UI updates on main thread

## CI Stability

### macOS Disk Space
- Added `Free disk space` step to all desktop build jobs (removes dotnet/Android/ghcActions, frees ~5GB)
- Disabled Go module cache for build jobs (`cache: false`), reducing ~1GB overhead
- macOS build script: deletes single-arch binaries after lipo, cleans staging after DMG packaging, runs `go clean -cache`
- Linux build script: runs `go clean -cache` after compilation
