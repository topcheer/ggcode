## v1.3.31

### Reasoning (Thinking) End-to-End Streaming

LLM reasoning/thinking tokens now stream through the tunnel to mobile with dedicated protocol events, symmetric to text streaming. Mobile shows expandable "Thinking" cards with markdown rendering, streaming dot indicator, and auto-collapse when the assistant response arrives.

- **Protocol**: `reasoning` / `reasoning_done` (main agent), `subagent_reasoning` / `subagent_reasoning_done` (sub-agents)
- **State boundaries**: Reasoning is finalized before every text, tool_call, tool_result, error, and done event — never crosses content boundaries
- **Redacted thinking**: `__redacted_thinking__` normalized to "Reasoning hidden by model."
- **Sub-agent reasoning**: Per-agent message ID derivation, automatic finalization on status transitions
- **Session resume**: Async loading with `agent.Clear()` before restoring messages

### Relay Stats Endpoint

New `/stats` HTTP endpoint with atomic counters for monitoring:
- Per-room event/client counts, total rooms, uptime
- Token masking for privacy (first 8 chars only)

### Flutter Workspace Cache SQLite Migration

Replaced SharedPreferences-based workspace cache with SQLite for better performance and reliability:
- Single transaction batch writes
- Error recovery: flush failure re-enqueues dirty keys
- Prune via SQL instead of in-memory map truncation
- Structured error reporting via `FlutterError.reportError`
