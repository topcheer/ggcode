# ggcode v1.1.79

## Highlights

Slash command Tab completion now fills the command into the input with a placeholder hint, instead of executing immediately.

## Changes

### Slash command Tab completion redesign
- Commands that accept optional arguments (`/harness`, `/mode`, `/model`, `/knight`, `/im`, `/config`, etc.) now **fill into the input** on Tab instead of executing immediately.
- A gray placeholder hint (e.g. `<subcommand>`) is shown after the cursor.
- Press Enter = execute without arguments. Continue typing = add arguments.
- Commands without arguments (`/help`, `/clear`, `/exit`, `/quit`, `/sessions`, `/plugins`, `/status`) still execute immediately on Tab.
- 17 commands with optional arguments mapped in `SlashCommandPlaceholders`.
