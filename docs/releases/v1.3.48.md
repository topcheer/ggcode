# v1.3.48

## Features

### Dynamic Terminal Window Title

The terminal window title now dynamically reflects the current state of ggcode:

- **Busy**: `> {activity} — {workspace} [{model}]` (e.g., `> Searching code — myproject [glm-5.1]`)
- **Idle**: `> ggcode — {workspace} [{model}]` (e.g., `> ggcode — myproject [gpt-5.4]`)

Updates on every TUI state change with deduplication to avoid unnecessary escape sequence writes. Control characters are stripped to prevent terminal injection. Title resets cleanly on exit.

### Commits (since v1.3.47)

```
feat(tui): dynamic terminal window title
```
