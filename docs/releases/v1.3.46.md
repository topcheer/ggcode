# v1.3.46

## Features

### Turn Metrics Digest

After each agent turn completes, a compact inline digest now appears in the chat area showing performance metrics for that turn:
- **TTFT** (Time To First Token)
- **Duration** (total turn time)
- **Think Time** (reasoning duration)
- **Tool Count** and **Slowest Tool** with duration
- **Failure indicator** when tools failed

The digest is deduplicated — it only appears once per turn, even if the UI re-renders. Available in both TUI and Desktop.

### Sidebar Default Hidden

- Sidebar now defaults to **hidden** on first launch for a cleaner initial experience.
- A `Ctrl+R sidebar` hint appears in the composer when the sidebar is hidden.
- Existing users' preferences are preserved — this only affects new installations.

## Improvements

### Collector Flush
- New `Flush()` method on metrics collector ensures all queued events are persisted before the turn digest is computed, guaranteeuring accurate numbers.
- `stopOnce` prevents double-close panic if `Stop()` is called multiple times.

### Pre-compact Timing Fix
- Background pre-compact is now deferred after the `agentDoneMsg` signal, preventing its own metrics from contaminating the just-completed turn's digest.
