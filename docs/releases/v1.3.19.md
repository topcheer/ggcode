# ggcode v1.3.19

## Highlights

- **Mobile connection feedback**: When a mobile client connects via tunnel, the QR code overlay auto-closes and a "Mobile connected" system message appears in chat.
- **Manual workspace URL entry**: The mobile scanner screen now supports pasting a workspace URL directly, in addition to QR scanning.
- **DingTalk Markdown messages**: DingTalk adapter now sends replies in Markdown format instead of plain text, preserving formatting from agent responses.

## Fixes and improvements

- **Discord community links**: Added Discord invite link across README (EN/CN), npm/Python packages, Desktop app tunnel info, and TUI QR overlay.
- **Relay server client notification**: `ggcode-relay` now notifies the server when a client connects, enabling real-time connection feedback.
- **A2A Token validator enhancements**: `NewTokenValidator` now requires a non-empty issuer URL and supports `WithValidIssuers` and `WithHMACSecret` options for more flexible token validation.
- **Mobile i18n**: Added `workspace.manual_hint`, `workspace.url_placeholder`, `workspace.connect_direct` translations in EN and zh-CN.
