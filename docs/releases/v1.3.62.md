# v1.3.62

TUI 多 agent 流式消息节流优化。

## Fixes

### TUI 多 agent 消息节流
- Sub-agent 流式 text/reasoning 现在按 80ms 批次聚合（~12.5Hz），不再每个 token 发一条消息
- Swarm teammate 流式 text/reasoning 同样 80ms 批次聚合
- 3 个 sub-agent + 4 个 teammate 并发时，消息量从 ~1400/s 降至 ~90/s
- 消除多 agent 并发时的 TUI 卡顿

### 其他
- 移除 `sendProgramMsgs` 的 `safego.Go` 包装（`program.Send` 本身线程安全，goroutine 只增加调度开销）

## Upgrade notes

无需特殊升级步骤。
