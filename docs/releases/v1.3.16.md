# v1.3.16

## Desktop i18n Improvements

- **Language normalization**: New `normalizeLanguage()` function handles zh/zh-CN/zh_Hans/zh-Hant → zh-CN mapping
- **Language persistence**: `config.go` now stores and loads `Language` field from config file
- **Dark theme contrast**: `isDarkElement` helper improves text contrast on dark backgrounds
- **Translation updates**: Complete en.json and zh-CN.json coverage for all UI strings

## Tunnel Protocol Enhancement

- **Cron message display**: New `display_text` and `kind` fields in tunnel protocol allow cron-triggered messages to appear as system messages instead of user messages
- **Protocol tests**: Added `TestBrokerPushUserMessageData` and related test coverage

## Mobile Chat Improvements

- **Send while running**: Input bar is always enabled — send messages while agent is processing
- **Cron message rendering**: Cron-triggered prompts now display as system messages with custom display text
- **Protocol sync**: `display_text` / `kind` fields added to mobile protocol model

## iOS Fastfile Fix

- Fixed `Shellwords.shelvescape` → `Shellwords.shellescape` typo (5 occurrences)
