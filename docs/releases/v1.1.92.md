# ggcode v1.1.92

## Highlights

- **Brewing timer** — Input panel now shows elapsed time since you sent a message, displayed in cyan at the end of the hints line (e.g. `酿造 12s` / `brewing 12s`). Resets on each new message.

## Fixes and improvements

- **TUI**: Brewing timer in composer panel shows agent loop duration in cyan.
- **TUI**: Timer resets on each new user message; disappears when agent finishes.
- **IM**: `edit_file` result now shows correct `+N -M` line counts parsed from rawArgs (was always `+0 -0`).
- **IM**: `write_file` result now shows correct line count from content arg (was always 1 line).
- **npm/PyPI README**: Added native installer download table linking to GitHub Releases; added feature highlights.

## Upgrade notes

- No config changes required.
- No breaking API changes.
