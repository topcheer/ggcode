# ggcode v1.3.70

## Highlights

- **Desktop message rendering fix**: Tool-use rounds now correctly display as separate assistant messages. Previously all LLM iteration text was merged into the first assistant message, making interleaved text invisible.
- **Tmux enhancements**: New `/tmux` actions â€” `logs`, `stop`, `rerun`, `delete_layout`, `rename_layout` â€” plus automatic layout inference from project files and `--setup` flag for quick session bootstrapping.
- **Cancel confirmation in TUI**: Pressing Ctrl-C or Esc during agent execution now shows a confirmation prompt instead of immediately cancelling, preventing accidental interruption.
- **Tunnel active-session barrier**: Mobile reconnect now waits for a barrier ordinal before marking the session ready, preventing partial data display after reconnect.
- **Desktop turn-based streaming**: Stream events now carry `turn_id`, `message_id`, and `seq` for precise frontend message matching and incremental rendering.
- **Desktop clipboard attachments**: Paste images from clipboard on macOS directly into chat.

## Fixes and improvements

- Fixed desktop assistant message merging across LLM iterations â€” each tool-use round now gets its own message bubble.
