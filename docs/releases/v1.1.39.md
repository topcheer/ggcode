# ggcode v1.1.39

## Fixes

- **Windows smoke test: Unblock-File to bypass Defender hang** — `Unblock-File` removes the NTFS Zone.Identifier alternate data stream that marks the binary as "downloaded from the internet", which was triggering Windows Defender to scan and hang `ggcode.exe` for 30+ minutes on first run in CI. Previous attempts using `Add-MpPreference` and `Set-MpPreference` silently fail on non-admin GitHub Actions runners.
