# ggcode v1.2.3

## Highlights

- **Working directory in sub-agent & teammate prompts**: Sub-agents and swarm teammates now receive the project working directory in their system prompts, eliminating the costly discovery phase (pwd → ls → find) that previously wasted multiple agent turns before real work could begin.
- **Cascade cancel on interrupt**: Pressing ctrl+c/esc or exiting (ctrl+d, double ctrl+c) now cancels all running sub-agents and swarm teammates, not just the main agent. No more orphaned background work consuming tokens.
- **Arrow key navigation for follow strip**: Up/down/left/right arrow keys now navigate between sub-agent and teammate slots in the follow panel (previously dead code — arrow key handling was unreachable).

## Fixes and improvements

- **Follow strip grace period**: Completed sub-agents remain visible in the TUI follow strip for 1 minute before being removed, giving users time to review results instead of slots disappearing instantly.
- **Tool label truncation fix**: `send_message` and `write_command_input` tool labels previously used byte-level truncation (`len(msg) > 40`), which corrupted multi-byte Unicode characters (Chinese, markdown bold `**`). Now uses visual-width-aware truncation via `runewidth`.
- **swarm_task_create display fix**: Long teammate role descriptions no longer overflow the tool header. `DisplayName` now uses only the first line of the description, with `assignee: subject` shown separately in the detail field.
- **swarm_task_create description emphasizes assignee**: Tool description now strongly recommends always setting the `assignee` field. Only leave empty when genuinely unable to determine who should do the task.
- **send_message clarifies role**: Description now notes that `swarm_task_create` is preferred for tracked task assignment, while `send_message` is for unstructured follow-ups and clarifications.
- **Ctrl+N hint in input area**: When sub-agent or teammate slots exist but follow mode is not active, the input hints area now shows a "Ctrl+N agents" hint.
- **Comprehensive test coverage**: All new functionality covered by tests — `CancelAll` for both sub-agents and swarm, `shutdownAll`, grace period filtering, `firstLine` helper, and system prompt working directory injection.

## Upgrade notes

- No breaking changes. All changes are backward-compatible.
- Sub-agents and teammates will immediately benefit from the working directory injection — no configuration changes needed.
