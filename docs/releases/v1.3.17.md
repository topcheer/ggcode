# v1.3.17

## Theme System (Desktop + Mobile)

- **6 color themes**: Midnight (deep blue, default), OLED Black, Nord (arctic blue), Rose (warm pink), Forest (emerald green), Light (day mode)
- **Desktop theme menu**: View → Theme submenu with check marks, instant switch without restart
- **Mobile theme button**: Palette icon in AppBar cycles through all 6 themes
- **Bidirectional sync**: Theme changes on either device propagate to the other via tunnel protocol
- **WCAG AA contrast tests**: All 6 themes validated — 4.5:1 for normal text, 3.0:1 for large text across 12 color pairs per theme
- **Zero migration cost**: Mobile `AppColors.xxx` remains static accessors (134 call sites unchanged), backed by mutable palette instance
- **Desktop palette architecture**: 21 color tokens per theme, Fyne `Theme` interface auto-refreshes all widgets on switch

## i18n Internationalization (Desktop + Mobile)

- **Bidirectional language sync**: Language changes on mobile ↔ desktop propagate in real-time via tunnel
- **Desktop (Fyne)**: `go:embed` JSON translations + `t()` function, live UI refresh without restart
  - Menus (File/View/Tools/Help), About dialog, Onboard form, Sidebar labels/forms, IM window, Share panel, File preview, Folder picker
- **Mobile (Flutter)**: `languageProvider` + `loadTranslations()` + `t()` function
  - Input bar hints, status labels, approval sheet, connect screen, chat screen, workspace switcher
  - Language toggle button in AppBar (EN / 中)
- **55+ translation keys** per language (English + Simplified Chinese)
- **Tests**: Desktop validates JSON validity + required keys; Mobile validates no duplicate keys

## Tunnel Protocol

- `EventThemeChange` / `CmdThemeChange` / `ThemeChangeData` for theme sync
- `EventLanguageChange` / `CmdLanguageChange` / `LanguageChangeData` for language sync
- Relay forwards both events to all other peers in room
- `relayMessage.Data` (json.RawMessage) field added for structured event payloads
