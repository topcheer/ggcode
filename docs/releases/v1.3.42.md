# v1.3.42

## Features

### XiaoMi MIMO Provider

- **New vendor**: `xiaomi-mimo` with two endpoints:
  - `cn-openai` — OpenAI protocol, `https://token-plan-cn.xiaomimimo.com/v1`
  - `cn-anthropic` — Anthropic protocol, `https://token-plan-cn.xiaomimimo.com/anthropic`
- **8 models**: MiMo-V2.5-Pro, MiMo-V2.5, MiMo-V2.5-TTS-VoiceClone, MiMo-V2.5-TTS-VoiceDesign, MiMo-V2.5-TTS, MiMo-V2-Pro, MiMo-V2-Omni, MiMo-V2-TTS
- **Capabilities**: V2.5 and V2-Omni infer vision support; 1M context window for main models
- **API key**: `XIAOMI_MIMO_API_KEY` env var

### TUI Sidebar Width

- **3-tier width**: terminals ≥180 columns get sidebar width 38, ≥140 get 36, default 34. Previous was 40/44 which felt too wide.
