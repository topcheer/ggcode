# ggcode v1.1.80

## Highlights

Fix @mention file completion: empty prefix now correctly lists working directory contents instead of parent directory. Directory selection auto-opens next level.

## Changes

### @mention completion fix
- **Root cause**: `CompleteMention("")` set `prefix = "."` then computed `filepath.Dir(workDir)` = parent directory, `filepath.Base(workDir)` = directory name. So typing `@` showed entries from the **parent** directory matching the current directory name.
- **Fix**: Empty prefix now directly reads `workDir` contents with no partial filter, showing all files/dirs in the current project.
- **Directory navigation**: Selecting a directory (e.g. `internal/`) now inserts `@internal/` without trailing space and re-triggers autocomplete to show its contents. User can drill down through the file tree.
- **File selection**: Selecting a file inserts `@filename ` with trailing space and closes the dropdown.
- Added `TestCompleteMentionEmptyPrefix` to prevent regression.
