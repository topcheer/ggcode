# ggcode v1.3.3

## Highlights

- **Linux desktop builds** — ggcode-desktop now ships for all three platforms: macOS (DMG), Windows (EXE), and Linux (amd64 binary). The Linux build runs on Ubuntu with Fyne dependencies pre-installed.
- **Sub-agent/teammate tab recycling** — Completed sub-agent and teammate follow-strip tabs are now properly cleaned up after a grace period, preventing indefinite clutter.

## Fixes and improvements

- `subagent/manager.go`: call `notifyUpdate()` in `Complete()` so TUI follow strip starts grace-period timer
- `swarm/idle_runner.go`: set `EndedAt` when teammate finishes a task for proper slot cleanup
- Desktop: session list item height fixed with padding wrapper
- Desktop: RichText wrapping enabled for tool headers and agent panel headers
- CI: Linux desktop build job with Fyne system dependencies
- CI: Windows desktop build uses absolute output paths (fixes empty artifact)
- CI: Artifact upload runs before release upload (prevents missing files)
- CI: Go module cache enabled across all workflows
