# v1.3.58

Desktop UI improvements.

## Desktop

### Session Switch Loading Effect
- Chat area clears immediately when switching historical sessions, showing "Loading session..." with animated ellipsis indicator
- Long operations moved to background goroutine, UI stays responsive

### Streaming Tool Call Display
- Tool call card shows spinner during streaming, displays tool name and description
- Auto-expands to show real-time argument text on stream updates
- Spinner stops when result arrives, card collapses to compact mode

### Chat Area Rendering Optimization
- MarkdownWidget main container VBox spacing from default 8px to 1px
- Paragraph/heading blocks use `compactRichText()` negative padding to counteract Fyne InnerPadding
- Table cell padding from 8px to 2px vertical + 4px horizontal
- Code block/quote block/list block internal spacing uniformly reduced
