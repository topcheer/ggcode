# ggcode v1.1.86

## Highlights

- **WeCom (Enterprise WeChat) IM adapter** — full AI Bot WebSocket gateway integration with pairing, inbound parsing, and outbound delivery
- **Mattermost IM adapter** — REST API v4 + WebSocket real-time events, threaded replies, @mention policy
- Both adapters verified end-to-end with live message exchange

## Fixes and improvements

### WeCom (Enterprise WeChat) — new platform

- AI Bot WebSocket protocol: `aibot_msg_callback` inbound, `aibot_send_msg` proactive outbound
- `aibot_respond_msg` stream reply: shows reply bubbles in WeCom UI when responding to tracked inbound messages
- Pairing flow: first inbound message triggers challenge-response
- Message parsing: text, markdown, images (base64), file attachments, quote extraction
- Access policy: DM/group allowlist, per-adapter `dm_policy` / `group_policy`
- Typing indicator: sends intermediate `respond_msg` with `finish=false`
- TUI panel (`/wecom`): full CRUD — create bot, bind/unbind workspace, edit config, i18n (EN + ZH)

### Mattermost — new platform

- REST API v4 for outbound (`POST /api/v4/posts`), WebSocket for real-time inbound (`posted` events)
- Auth: Bearer token (personal access token or bot token)
- @mention policy: channel messages require @mention unless in `free_channels`; DMs exempt
- System message filtering (join/leave/header_change etc.)
- Typing indicator: WebSocket `user_typing` action
- TUI panel (`/mattermost`, `/mm`): full CRUD with i18n (EN + ZH)

### Integration & UX

- Slash commands `/wecom`, `/mattermost`, `/mm` registered in autocomplete, help text, and dispatch
- Platform display names in sidebar (`WeCom`/`企业微信`, `Mattermost`)
- Paste events intercepted in create/edit modes for both panels
- Result message handlers wired into TUI event loop

## Upgrade notes

- No breaking changes — existing adapters and config are unaffected
- New adapters are opt-in via config or TUI panel
- WeCom requires an AI Bot application registered in the WeCom admin console
- Mattermost requires a personal access token or bot account with API access
