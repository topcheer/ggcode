# ggcode v1.3.18

## Fixes and improvements

- **Streaming markdown table rendering**: Table nodes were split into individual rows/cells during streaming, losing pipe characters (`|`) and separator lines (`| --- |`). Tables are now treated as atomic blocks (like lists and blockquotes), preserving full markdown syntax in the TUI chat view.
