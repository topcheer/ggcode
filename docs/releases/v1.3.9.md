# ggcode v1.3.9

## Fixes and improvements

- **TUI**: `/share` and `/tunnel` commands now execute immediately even when the agent is busy, instead of being queued
- **Makefile**: added `store-deploy`, `store-deploy-ios`, `store-deploy-android`, `store-version`, `store-tag` targets for one-command mobile app releases
- **Mobile**: end-to-end App Store / Google Play deployment pipeline:
  - `store_deploy.sh` — automated build, upload, and promote for both platforms
  - `version_sync.sh` — syncs version across pubspec.yaml, build.gradle.kts, and Info.plist
  - Fastfile lanes: `upload_testflight`, `deploy_external`, `upload_metadata`, `submit`, `inspect`
  - Real simulator screenshots (iPhone 6.7" + iPad 13") for App Store listing
